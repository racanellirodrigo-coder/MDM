# Changelog

## [1.0.0] - 2026-05-03

### ✨ Features
- ✅ App completo de orçamento de móveis
- ✅ Cálculo automático por metragem
- ✅ Gerenciamento de preços de materiais
- ✅ Show room com galeria de projetos
- ✅ Interface luxuosa Gold Edition
- ✅ Login seguro com autenticação
- ✅ Suporte a múltiplos tipos de móveis

### 🎨 Design
- Interface elegante com tema dourado
- Animations de loading
- Layout responsivo
- Componentes nativos Android

### 🔧 Técnico
- Python 3.11
- Kivy 2.3.0
- Pillow para processamento de imagens
- JSON para armazenamento de dados
- API Android 31 (compatível 21+)

### 📱 Especificações
- Tamanho: ~50MB
- Arquiteturas: ARM64 + ARMv7
- Permissões: INTERNET, STORAGE
- Orientação: Portrait
